ECMA klasser/konstruktører

ECMA inkluderer en mengde nyttige klasser, eller konstruktørfunksjoner.

JavaScript er ikke klassebasert, og klassene er funksjoner. I JavaScript kan funksjoner brukes sammen med operator new og da vil funksjonen brukes som en konstruktør for å lage et objekt. I praksis kan derfor ECMA sine konstruktørfunksjoner ses på som klasser.
