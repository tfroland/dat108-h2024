JSON objektet

All utskrift fra JavaScript-koden skrives til web konsollet.

Objektet JSON inkluderer funksjoner for å konvertere mellom JavaScript objekt og tekst i JSON syntaks.

JSON objektet er nytt f.eks. når webapplikasjon skal utveksle data mellom nettleser og webtjener.

JSON står for JavaScript Object Notation.
